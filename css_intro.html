What is CSS?

CSS, or Cascading Style Sheets, is a style sheet language used for describing the look and formatting of a document written in HTML. CSS is designed to enable the separation of presentation and content, including layout, colors, and fonts.

How CSS styles HTML documents

CSS styles are applied to HTML elements using selectors. A selector can be an element name, a class name, an ID, or an attribute among others. The CSS rules for a selector are enclosed in curly braces {}.

Basic CSS properties for styling

There are many CSS properties for styling HTML elements. Some common ones include color for text color, background-color for background color, font-size for text size, border for borders, margin for space outside elements, and padding for space inside elements.
